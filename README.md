# erase-indent

- [View documentation generated by TypeDoc](https://sakkke.github.io/erase-indent/)
- [View code coverage report by Jest](https://sakkke.github.io/erase-indent/_coverage/lcov-report/)

## Usage

```javascript
import { Eraser } as '@0wv/erase-indent'
const eraser = new Eraser(`  def hello():
    pass`)
eraser.erase()
console.log(eraser.string) /* `def hello():
  pass` */
```
